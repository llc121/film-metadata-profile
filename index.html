Introduction

This metadata schema describes a curated collection of film memorabilia, including posters, scripts, autographs, and ephemera. It draws on established metadata standards to ensure interoperability while introducing custom fields for domain-specific features like autographs and restoration notes.

The schema is designed for private collectors, archives, and digital collections needing rich descriptive metadata for analog and digital audiovisual items.

Metadata Standards Used

  • Dublin Core (dc): Basic descriptive elements
  • PBCore (pbcore): Audiovisual-specific metadata
  • MODS (mods): Bibliographic structure
  • Schema.org (schema): Web-structured data
  • EAD (ead): Archival structure and arrangement
  • LCSH (lcsh): Subject vocabularies
  • OAI-PMH (oai): Harvesting compatibility
  • FMC (fmc): Custom namespace for film-specific needs
